\# Bank Account Management System (Java)



A simple Java portfolio project that models bank account operations and applies transactions from a file.



\## Features

\- Deposit and withdraw operations

\- Custom exception: InsufficientFundsException

\- Transaction processing from an input file

\- Logging of results to an output file

\- Separation of business logic and file I/O



\## How to Run

Run the `Main` class in IntelliJ IDEA.



\## Example

Input file contains transactions like:

DEPOSIT;ACC-1001;250.00

WITHDRAW;ACC-1001;80.50
